# カスタム歴史タイムライン

歴史イベントとゲームイベントを組み合わせて表示するタイムライン生成ツール。

テキストエリアに `日付 | タイトル | 説明` の形式で入力すると、左右交互に配置されたタイムラインを生成する。タイトルの先頭に★を付けると比翼イベント（桜エフェクト）、✕を付けると破局イベント（雪エフェクト）として表示される。

## デモ

https://mayholiholi.github.io/timeline-generator/

## 入力例

```
1853年7月 | 黒船来航 | ペリー提督が浦賀に来航
1864年 | ★坂本龍馬と比翼 | 隠し刀との絆
1865年 | ✕坂本龍馬と破局 | 二股がバレたため
```
